This project implements a complete ML pipeline for environmental sound classification using the UrbanSound8K dataset. The system can:

2. Car Horn

4. Dog Bark
5. Drilling
6. Engine Idling
7. Gun Shot
8. Jackhammer
9. Siren
10. Street Music
